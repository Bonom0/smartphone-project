# smartphone-project
Projeto Final da matéria de Ciência de Dados

Projeto de Análise de Dados – SIM BigData

Este projeto realiza uma análise exploratória de dados usando Python no Google Colab, utilizando uma base de dados com informações diversas. O objetivo é aplicar conceitos de Data Science para limpeza, transformação, análise estatística e visualização dos dados.

Tecnologias Utilizadas
  Python
  Google Colab
  Pandas
  NumPy
  Matplotlib

Etapas do Projeto
  Importação dos dados pelo Colab
  Pré-processamento: correção de tipos, remoção de inconsistências e criação de filtros
  Análise exploratória
  Geração de gráficos (distribuições, comparações e correlações)
  Conclusões com base nos resultados obtidos
